# argolizm

クリエイティブ職（エンジニア・デザイナー等）の市場価値を算定するオープンアルゴリズム。

PDF職歴書・GitHub・ポートフォリオを入力すると、技術力・実績・成長性の3軸でスコアを算出し、推定月額単価レンジを表示します。

> **⚠️ 現在はプロトタイプです。** スコア計算はモック実装です。

---

## スクリーンショット

```

│  [ドロップゾーン]          │    │                          │
│  [抽出テキスト preview]   │    │  技術力  ████████░░  80  │
│                          │ →  │  実績    ███████░░░  70  │
│  2. GitHub               │    │  成長性  █████████░  85  │
│  [username 入力]          │    │                          │
│  [プロフィールカード]      │    │  市場価値: ¥70万〜¥90万  │
│                          │    └──────────────────────────┘
│  3. ポートフォリオ         │
│  [URL × 最大5件]          │
│                          │
│  [スコアを算出する]        │
└──────────────────────────┘
```

## 機能

| 機能 | 概要 |
|---|---|
| PDF 職歴書解析 | drag&drop でアップロード。`pdfjs-dist` によりブラウザ内でテキスト抽出（サーバー送信なし） |
| GitHub 連携 | ユーザー名を入力して GitHub REST API からパブリックプロフィールを取得 |
| ポートフォリオ入力 | URL を最大5件登録 |
| スコア算出 | 技術力・実績・成長性の3軸 + 総合スコア + 推定月額単価レンジ |

---

## スコアリングアルゴリズム（`src/lib/scorer.ts`）

```
総合スコア = (技術力 + 実績 + 成長性) / 3

技術力:  base 40 + GitHub repos数 (max +25) + PDF内キーワード (max +15) + ポートフォリオ (+5)
実績:    base 40 + PDF職歴書あり (+20) + ポートフォリオURL数 (max +20)
成長性:  base 40 + GitHub followers数 (max +15) + repos > 20 (+5)
```

> このファイルが将来 npm パッケージとして OSS 公開される予定です。

## セットアップ

```bash
git clone https://github.com/tsubasagit/talent-argolizm.git
cd talent-argolizm
npm install
npm run dev
# → http://localhost:3003
```

Node.js 18以上が必要です。

## 技術スタック

- **Next.js 16** (App Router, TypeScript, Turbopack)
- **Tailwind CSS v4**
- **pdfjs-dist v4** — クライアントサイド PDF テキスト抽出
- **GitHub REST API** — パブリックプロフィール取得（認証不要）
